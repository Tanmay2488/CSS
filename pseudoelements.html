CSS Pseudo-elements Example

This paragraph demonstrates the use of pseudo-elements in CSS. The content before and after the paragraph is added using ::before and ::after, respectively. The first line of the paragraph is styled differently using ::first-line.

CSS pseudo-elements allow you to style specific parts of an element, such as the first line or adding content before or after an element.
